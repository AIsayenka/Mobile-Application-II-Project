Detailed Description

Represents an account owned by a user.

Remarks
If the agent is authenticated via the OAuth Reigstered application workflow (with no known user), then the consumer_key should be used as the accountName, and the token request endpoint should be used as the accountServiceHomePage. It is crucial not to do this for any agent object where a known user is involved, as this would lead to logically different agents being considered identical by the LRS.
