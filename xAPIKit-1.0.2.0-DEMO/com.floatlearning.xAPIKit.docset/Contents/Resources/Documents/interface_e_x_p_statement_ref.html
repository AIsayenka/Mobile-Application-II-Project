Detailed Description

Represents a StatementRef.

A Statement Reference is a pointer to another pre-existing Statement.

A Statement Reference MUST specify an "objectType" property with the value "StatementRef". A Statement Reference MUST set the "id" property to the UUID of a Statement which is present on the system.

StatementRefs are used when voiding statements.
