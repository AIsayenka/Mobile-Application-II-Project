Changelog

1.0.0.1

  • The library has been renamed to reflect the official name of the API: the Experience API (xAPI).
  • All class names have been renamed from "TC…" to "EXP…". For the most part, simply search your code for "TC" and replace it with "EXP".
  • The capitalization of the word 'Id' has now been standardized to 'Id'. Method names such as "activityWithID:" have been renamed to "activityWithId:".
