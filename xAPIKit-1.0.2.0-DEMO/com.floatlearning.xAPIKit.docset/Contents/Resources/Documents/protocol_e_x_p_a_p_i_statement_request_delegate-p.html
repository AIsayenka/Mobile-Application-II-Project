Detailed Description

Extended delegate methods for statement requests.

Notified when statements are received from the LRS, when they are stored on the LRS, or when a failure was encountered while trying to persist statements on the LRS. Each of these come before requestDidFinish:

Method Documentation

- (NSURL *) resolveURLFromMoreLink: (NSString *)  moreLink
optional

There is an inconsistency with how the More link is generated from different LRS's.

As the creation of the 0.95 library, this inconsistency has yet to be resolved. In response, the library is offering a delegate method to allow implementations to customize this. If you implement this method, you should return the full URL for where the next request should be made.

Parameters
moreLinkThe URL/path received from the last statement result.
Returns
The resolved URL for the next request.

1.0
