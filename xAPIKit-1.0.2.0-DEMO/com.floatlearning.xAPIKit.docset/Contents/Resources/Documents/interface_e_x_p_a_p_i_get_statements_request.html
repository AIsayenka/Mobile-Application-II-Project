Initializes a request to the LRS to continue to retrive statements from a previous request.

The result EXPStatementResult will also contain the statements from the previous result.

Parameters
aResultThe previous EXPStatementResult.
aURLThe URL to the LRS (not to the endpoint for the API call–the endpoint it automatically appended).
aAuthProviderThe class that provides the authorization instructions for the class.
aDelegateThe delegate for the request.
Returns
The initialized EXPAPIRequest to obtain statements.
