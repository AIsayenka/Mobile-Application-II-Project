Detailed Description

Adds support for adding query parameters to a URL (e.g.

?property=value).

Method Documentation

- (NSURL *) URLByAppendingQueryString: (NSString *)  query

Returns a new URL made by appending a prepared query string to the original URL.

Parameters
queryThe query string to append to the URL (e.g ?property=value).
Returns
A new URL with query appended.
Remarks
If the query does not begin with a question mark (?) then one will be inserted at the beginning of the string before appending to the URL.
- (NSURL *) URLByAppendingQueryWithDictionary: (NSDictionary *)  query

Returns a new URL made by appending a generated query string to the original URL.

The query string is generated by the keys and values of the supplied dictionary.

Parameters
queryThe dictionary to grab the keys and values from to generate a query string.
Returns
A new URL with the generated query string appended.
